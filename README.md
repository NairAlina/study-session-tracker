# Study Session Timer & Tracker

## 📌 Description
This is a Java Swing application that helps users track their study sessions using a timer.

## 🚀 Features
- Start, Stop, Reset timer
- Enter subject and duration
- Session history tracking
- Total study time calculation
- Delete session option

## 🧠 OOP Concepts Used
- Classes and Objects
- Encapsulation
- Separation of concerns

## 🖥️ Technologies
- Java
- Java Swing (GUI)

## ▶️ How to Run
Run Main.java to start the application.
