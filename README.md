# Study Session Timer & Tracker

## 📌 Description
This is a Java Swing application that helps users track their study sessions using a timer.

## 🚀 Features
- Start, Stop, Reset timer
- Enter subject and duration
- Session history tracking
- Total study time calculation
- Delete session option

## 🧠 OOP Concepts Used
- Classes and Objects
- Encapsulation
- Separation of concerns

## 🖥️ Technologies
- Java
- Java Swing (GUI)

## ▶️ How to Run
Run Main.java to start the application.

## 🚀 How to Run the Project

1. Download the project (ZIP) or clone using:
   git clone https://github.com/NairAlina/study-session-tracker.git

2. Open the project in VS Code or IntelliJ

3. Navigate to:
   src/com/studytracker/Main.java

4. Run the Main.java file
